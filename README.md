# Advent of code 2022

## About
OK, here we go again!

https://adventofcode.com/2022

## Quickstart
```
python3 -m venv venv
. ./venv/bin/activate
pip install -r requirements.txt
python aoc.py --day "01"
```

## Convenience tools
### Create the base files for a new day
```
make prepare day=03
```
### Run a solution for the test input
```
. ./venv/bin/activate
python aoc.py --day "01" --test
```
